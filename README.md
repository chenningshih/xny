java+vue新能源汽车销售平台(计算机毕设)

项目有
客户端：
登录注册，用户信息修改，首页，热门车型展示，车辆列表，购物车，支付订单，商品收藏，售后（维修及退款），客服（deepseek智能客服，在线人工客服）
管理端：
登录，用户信息修改，车辆列表，车辆分量，车辆库存管理，订单列表，订单处理，售后处理，客户信息，人工客服，以及数据统计页面  提供源码和开发文档
